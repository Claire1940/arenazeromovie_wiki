## Mécaniques de bataille : Hitboxes et slime de guérison

Le combat dans Arena Zero est régi par une physique d'IA stricte, bien que parfois « buggée ». Les spectateurs remarquent souvent que les personnages se plaignent de « hitboxes atroces ». En termes de jeu, une hitbox est la forme invisible qui détermine si une attaque touche une cible. Dans la série, ces hitboxes sont dynamiques, ce qui signifie que les joueurs doivent être précis dans leurs mouvements.

La guérison est une autre mécanique critique. Lorsque Leo a été blessé lors de son premier combat, Hokei a appliqué un « slime de guérison ». Cette substance agit temporairement pour arrêter le saignement et restaurer les HP, mais ce n'est pas une solution permanente. Elle sert de « tampon » tactique pour maintenir le champion dans le combat jusqu'à la période de repos de 480 heures entre les matchs.

| Mécanique | Fonction dans Arena Zero | Valeur stratégique |
| :--- | :--- | :--- |
| **Hitboxes** | Détermine le contact physique | Élevée (nécessite de la précision) |
| **Slime de guérison** | Restauration temporaire des HP | Moyenne (arrête le saignement) |
| **Temps de recharge** | Période de repos de 480 heures | Essentiel pour la récupération |
| **Chronos** | Limite de match de 3 minutes | Force un jeu agressif |

> **💡 Conseil :** Surveillez toujours le chronomètre. Dans Arena Zero, les matchs sont conçus pour être rapides. Si un champion ne parvient pas à remporter la victoire dans le temps imparti, des dangers environnementaux ou des mécaniques de « mort subite » peuvent se déclencher.
